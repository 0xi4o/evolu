# API

Public API is documented with JSDoc.

Packages for app development:

- `@evolu/react`
- `@evolu/react-native`
- `@evolu/common-web` (All other Frameworks)

To define the Evolu database schema reusable for platforms, use `@evolu/common`.

If you want to host your own Evolu Server, use `@evolu/server`.
